BulkRenamer – Массовый переименователь файлов на 7 языках
Мощный инструмент для массового переименования файлов с гибкими шаблонами, регулярными выражениями, фильтрами и режимом предварительного просмотра.
Поддерживает рекурсивную обработку подпапок, замену текста, добавление префиксов/суффиксов, нумерацию, смену регистра и многое другое.
Реализован на семи языках программирования с единым интерфейсом командной строки.

🚀 Возможности
Шаблоны имён – используйте подстановки: {name}, {ext}, {num}, {date}, {old} и др.

Замена текста – замена части имени по точному совпадению или регулярному выражению.

Добавление префикса/суффикса – вставка текста в начало или конец имени.

Нумерация – автоматическая нумерация файлов с настраиваемым форматом (ширина, старт).

Смена регистра – приведение к верхнему, нижнему, заглавному регистру.

Фильтрация – по расширению, по маске, по размеру, по дате изменения.

Рекурсивный режим – обработка всех подпапок.

Режим симуляции (dry-run) – показать, что будет изменено, без фактического переименования.

Цветной вывод – информация об успехе/ошибке.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> <шаблон> [опции]
Основные опции
Опция	Описание
-p, --path <dir>	Корневая папка (по умолчанию текущая)
-r, --recursive	Рекурсивно обрабатывать подпапки
-e, --ext <ext>	Фильтр по расширению (можно несколько через запятую)
-f, --filter <regex>	Фильтр по имени (регулярное выражение)
--prefix <text>	Добавить текст в начало имени
--suffix <text>	Добавить текст в конец имени (перед расширением)
--replace <from> <to>	Заменить подстроку (можно использовать regex)
--case <lower|upper|title>	Привести регистр
--number <format>	Вставить номер, например {num:03d} (ширина 3)
--dry-run	Только показать изменения, не переименовывать
-v, --verbose	Подробный вывод
Шаблоны
В шаблоне можно использовать переменные:

{name} – исходное имя без расширения

{ext} – расширение (без точки)

{num} – номер (если указан --number)

{date} – текущая дата в формате ГГГГ-ММ-ДД

{old} – полное исходное имя

Примеры
bash
# Добавить префикс "photo_" ко всем JPG в папке
python bulk_renamer.py "{prefix}_{name}.{ext}" --prefix photo -e jpg

# Заменить "IMG_" на "image_" в именах
python bulk_renamer.py "{name}.{ext}" --replace "IMG_" "image_"

# Нумерация файлов с шириной 3, начиная с 1
python bulk_renamer.py "file_{num:03d}.{ext}" --number 1 -e txt

# Рекурсивно переименовать все .log, добавив дату
python bulk_renamer.py "{date}_{name}.{ext}" -r -e log

# Только показать, что будет изменено
python bulk_renamer.py "new_{name}.{ext}" --dry-run

🧠 Архитектура
Все реализации построены по единой логике:

Парсинг аргументов и опций.

Сбор списка файлов по указанным фильтрам (рекурсивно).

Для каждого файла вычисляется новое имя на основе шаблона и опций.

Если режим --dry-run – выводятся старые и новые имена.

Иначе выполняется переименование с проверкой конфликтов (если новое имя уже существует, добавляется суффикс).

✨ Дополнительные фичи
Защита от конфликтов – если новое имя уже занято, программа добавит числовой суффикс.

Поддержка Unicode – корректно работает с любыми символами.

Подробные сообщения – цветной вывод успешных операций и ошибок.
